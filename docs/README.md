# سیستم مدیریت انبار

## معرفی
سیستم مدیریت انبار یک راهکار جامع برای مدیریت موجودی، ردیابی کالاها، و کنترل انبار است. این سیستم به کسب‌وکارها کمک می‌کند تا انبارداری خود را بهینه کرده و روند‌های مرتبط را خودکارسازی کنند.

## ویژگی‌های اصلی
- مدیریت موجودی و کالاها
- ثبت و پیگیری ورود و خروج کالا
- مدیریت کاربران با سطوح دسترسی مختلف
- گزارش‌گیری پیشرفته
- پشتیبانی از تقویم شمسی
- رابط کاربری ساده و کاربرپسند
- پشتیبانی از دستگاه‌های مختلف (ریسپانسیو)

## نیازمندی‌های سیستم
- PHP 7.4 یا بالاتر
- MySQL 5.7 یا بالاتر
- Apache/Nginx
- مرورگر مدرن (Chrome، Firefox، Safari، Edge)

## نصب و راه‌اندازی

### پیش‌نیازها
1. یک سرور وب با پشتیبانی از PHP
2. دیتابیس MySQL
3. دسترسی به خط فرمان برای نصب

### مراحل نصب
1. فایل‌های پروژه را در مسیر وب‌سرور خود کپی کنید
2. دیتابیس جدید ایجاد کنید
3. فایل `config.example.php` را به `config.php` تغییر نام داده و اطلاعات دیتابیس را در آن وارد کنید
4. به آدرس `http://yourdomain.com/setup.php` مراجعه کرده و مراحل نصب را دنبال کنید
5. پس از نصب، می‌توانید با نام کاربری و رمز عبور ادمین وارد سیستم شوید

## ساختار پروژه
- `admin/`: فایل‌های مربوط به پنل مدیریت
- `assets/`: فایل‌های CSS، JavaScript و تصاویر
- `core/`: هسته اصلی سیستم
- `includes/`: فایل‌های PHP مشترک
- `themes/`: پوسته‌های قابل تغییر سیستم
- `docs/`: مستندات پروژه
